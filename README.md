# 🖥️ NFControl Desktop

Aplicação desktop desenvolvida em C# Windows Forms para protocolo e recebimento de notas fiscais.

Este projeto foi a primeira versão do processo de protocolo utilizado posteriormente na versão web do NFControl, sendo utilizado para registro inicial e acompanhamento operacional do recebimento de notas fiscais.

---

## 🛠️ Tecnologias Utilizadas

- C#
- .NET Framework
- Windows Forms
- MySQL

---

## 📌 Status do Projeto

Projeto legado mantido para fins de estudo, histórico de evolução da aplicação e demonstração de experiência com aplicações desktop.
